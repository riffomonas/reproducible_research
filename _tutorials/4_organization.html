An aside about large files * GitHub cannot easily handle large datasets * Each file is limited to 100 MB and a repository is limited to 1 GB * Generally only post the data files critical to an analysis * Scripts should document where data goes and how it gets there * We'll talk more about git/GitHub later --- ##
